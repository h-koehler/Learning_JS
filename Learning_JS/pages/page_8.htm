toString()

The toString() method returns a number as as string.

toExponential()

toExponential() method returns a string, with a number rounded and written using exponential notation. A parameter defines the number of characters behind the decimal point.

toFixed()

toFixed() return a string, with the number written with a specific number of decimals.

toPrecision()

The toPrecision() method returns a string, with a number written with a specified length.

valueOf()

The valueOf() method returns a number as a number. It is used internally to convernt Number objects to primitive values.
