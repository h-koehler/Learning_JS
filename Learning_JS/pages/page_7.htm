All string methods return a new string. They don't modify the original string.


String indexOf()

The indexOf(string) method returns the index (position) of the first occurance of a string in a string. Returns -1 if the text is not found. Can take a second start position argument.

String lastIndexOf()

lastIndexOf(string>) method reurns the index of the last occurance of a string in a string. Returns -1 if the text is not found. Can take a second start position argument.

String search()

searach(string) method search the string for a string or regular expression and returns the position of the match.

String match()

The match(string) method returns an array containing the results of matching against a string or regular expression.

String matchAll()

The matchAll(string) method returns an iterator containing the reuslts of matching a string against a string or a regular expression.

String includes()

The includes(value) method returns true if a string contains a specified value. Otherwise it returns false.

String startsWith()

The startsWith(value) method returns true if a string contains a specified value. Otherwise it returns false.

String endsWith()

The endsWith(value) method returns true if a string ends with a specified value. Otherwise it returns false.
