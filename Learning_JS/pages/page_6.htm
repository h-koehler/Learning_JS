All string methods return a new string. They don't modify the original string.


String length

The length property returns the length of a string.

String slice()

slice(start, end) extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: start position, and end position (end not included). If the parameter is negative, the position is counted form the end of the string.

String substring()

substring(start, end) extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: start position, and end position (end not included). If the parameter is negative, it is treated as 0.

String substr()

substr(start, length) extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: start position, length (length of the extracted part).

String replace()

The replace(value1, value2) method replaces a specified value with another value in a string.

String replaceAll()

The replace(value1, value2) method allows you to specify a regular expression instead of a string to be replaced.

String toUpperCase()

The toUpperCase(value) method converts a string to upper case.

String toLowerCase()

The toLowerCase(value) method converts a string to lower case.

String concat()

The toUpperCase(value1, value2, ..., valueX) joins two or more strings. It can be used instead of the plus operator.

String trim()

The trim() method removes whitespace from both sides of a string.

String trimStart()

The trimStart() method removes whitespace only from the start of a string.

String trimEnd()

The trimEnd() method removes whitespace only from the end of a string.

String padStart()

The padStart(length, string) method pads a string from the start. It pads a string with another string (multiple times) until it reaches a given length.

String padEnd()

The padEnd(length, string) method pads a string from the end. It pads a string with another string (multiple times) until it reaches a given length.

String charAt()

The charAt(position) method returns the character at a specified index (position) in a string.

String charCodeAt()

The charAt(position) method returns the unicode (UTF-16 code) of the character at a specified index in a string.

String split()

The split(separator) method converts a string into an array. If the separator is omitted, the returned array will contain the whole string in index[0]. If the separator is "", the returned array will be an array of single characters.
